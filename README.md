# Где дешевле жить в Казахстане

Веб-приложение для сравнения цен на продукты, топливо и расходов в различных городах Казахстана.

## Технологии

- **Next.js 14** - React-фреймворк
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **React Hooks** - Управление состоянием

## Структура проекта

```
kazakhstan-cost-comparison/
├── app/
│   ├── layout.tsx          # Корневой layout
│   ├── page.tsx            # Главная страница
│   └── globals.css         # Глобальные стили
├── components/
│   ├── Header.tsx          # Шапка с калькулятором
│   ├── QuickStats.tsx      # Быстрая статистика
│   ├── CategoryCards.tsx   # Карточки категорий
│   ├── PriceTable.tsx      # Таблица цен
│   └── Footer.tsx          # Подвал
├── public/                 # Статические файлы
├── package.json
├── tsconfig.json
├── tailwind.config.ts
├── postcss.config.js
└── next.config.js
```

## Установка и запуск

### Предварительные требования

- Node.js 18+ 
- npm или yarn

### Установка зависимостей

```bash
npm install
# или
yarn install
```

### Запуск в режиме разработки

```bash
npm run dev
# или
yarn dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена

```bash
npm run build
# или
yarn build
```

### Запуск продакшен версии

```bash
npm run start
# или
yarn start
```

## Функционал

- ✅ Калькулятор расходов по городам
- ✅ Сравнение цен на продукты
- ✅ Быстрая статистика
- ✅ Интерактивные таблицы
- ✅ Адаптивный дизайн
- ✅ Выбор города

## Компоненты

### Header
Содержит калькулятор жизни и средние показатели по городу.

### QuickStats
Отображает основную статистику проекта (количество городов, цен, частота обновлений).

### CategoryCards
Карточки основных категорий: Продукты, Топливо, Кредиты, Сравнения.

### PriceTable
Интерактивная таблица цен на продукты с выбором города.

### Footer
Подвал сайта с копирайтом.
